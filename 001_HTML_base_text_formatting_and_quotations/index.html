Text formatting (string tags!)

/b - simple bold text
/strong - strong (important) text
/em - emphasized text
/i - italic text
/small - this text is smaller then previous
/sub - subscripted text
/sup - supescripted text
/ins - underlined text
/del - a line through deleted text
/mark - marked text
